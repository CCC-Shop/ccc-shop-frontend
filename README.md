# ccc-shop-frontend

## Installation

### Install Yarn
```
npm install --global yarn
```

### Install dependencies
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Build production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

## Packages

- [Vue](https://vuejs.org/)
- [Vue Router](https://router.vuejs.org/)
- [Vuetify](https://vuetifyjs.com/)
